## Installation

### **NPM**

```bash
npm i restful-db
```

**TypeScript**

```typescript
import Database from 'restful-db'
```

**JavaScript**

```javascript
const { Database } = require('restful-db')
```

### **Browser**

```html
<script src="https://wzrd.in/standalone/restful-db"></script>
```

```javascript
const { Database } = restfulDb
```
